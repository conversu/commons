# COMMONS
Node tool kit used by Conversu.


## How to use

Install using npm or yarn:

```bash
npm i @conversu/commons
```

```bash
yarn add @conversu/commons
```

## TESTS
Run test suite:
```bash
npm run test
```

Run test suite and watch:
```bash
npm run test -- --watch
```

Generate coverage report:

```bash
npm run test -- --coverage
```
